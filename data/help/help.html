Collections

Create collection

Collection is a directory with a set of images. The collection can be created from an existing directory by choosing File >  Create new collection from the menu bar and setting a path to the collection.

Open collection

If you  have already created a collection, you can open an existing collection by choosing its name from the drop-down list in the section Collections.

Add image to collection

To add a single image to the current collection choose File >  Add image to collection from the menu bar. A new window will pop up, in which you can choose the path to the desired image.

Export an image

You can save the currently modified image by clicking Export > Export in the menu bar. In the next step a window will appear, where you will be able to choose the destination directory.

Export collection

You can save the currently modified image by clicking Export all in the menu bar. In the next step a window will appear, where you will be able to choose the destination directory. The whole collection will be saved in the chosen directory in the subdirectory named as the collection.

Remove current collection

To remove current collection choose File > Remove current collection from the menu bar. The directory and files will be preserved but history of changes will be deleted and collection will not be listed among active collections.

Remove files from collection

To remove images, firstly select images you want to remove in the Collections dock widget. Then, choose File > Remove selected images from the menu bar. The files will be preserved but history of changes will be deleted and the image will no longer appear in Collections dock widgets.

Save project

To save the current state of project choose File > Save project. It will preserve all changes to images, current collection and current image.

Effects

Straighten lines

In order to straighten the lines of text, choose either the radio button Straighten Lines in Effects  dock widget or click Edit > Effects > Straighten lines from the menu bar.  

Show lines

To better see the current position of letters, words and text lines you can add guide lines on both original image and modified image.

Main lines

These lines define the text line area with respect to the most borderline points.

Minor lines

These lines define the text line area referring to the innermost boundary.

Text block

These lines define a rectangle that encloses the entire block of text.

Words

These lines define rectangles enclosing separate words.

Letters

These lines define rectangles enclosing separate letters and punctuation marks.

Clean page

In order to clean the page, you can either click Edit > Effects > Clean page from the menu bar or in the Effects dock widget. In the first case a window with Light and Dark sliders will appear, after setting desired values click the Clean Page button and close the pop-up window. In the second case set preferred parameters on Light and Dark sliders in the Effects dock widget and click the Clean page button. On the Light slider you can adjust how strong the page light background will be smoothed out, whereas Dark slider value corresponds to strength of letters color smoothing.

Adjust contrast

To change the page contrast, you can either click Edit > Effects > Change contrast from the menu bar or in the Effects dock widget. In the first case a window with Intensity slider will appear, after setting desired value, click the Change contrast button and close the pop-up window. In the second case set preferred parameters on the Intensity slider in the Effects dock widget and click the Change contrast button.

Remove stains

In order to remove stains from your page use Remove Stains from the Effects dock widget. You can choose the size of the brush by using the slider. Then click the area with the stain you want to remove.

Apply effects to single  photo

After setting values on sliders in Effects dock widget, you can apply them all to the current image by clicking Apply in Effects dock widget or in the menu bar Edit > Apply.

Apply effects to whole collection

After setting values on sliders in Effects dock widget, you can apply them all to the whole collection by clicking Apply all in Effects dock widget or in the menu bar Edit > Apply all.

Reset effects

Revert the image state to the initial state. All changes are discarded. You can reset the image by clicking the reset button in the Effects dock widget or from the menu bar in section Edit > Reset.

Undo, redo effects

Undo - the button with an arrow pointing left on the toolbar. Erases last modification done to the image.

Redo - the button with an arrow pointing right on the toolbar. The opposite action to undo. Restores the last undone change.

Note: if buttons have a different background than the rest of the toolbar, it means you cannot perform this action (e.g. current image is the most recent version, you cannot redo anything).

App style

Change theme

In order to change the app theme, pick prefered theme from the section View in the menu bar. The new theme will be set after restarting the application.
